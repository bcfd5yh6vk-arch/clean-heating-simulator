# 保定农户清洁取暖转型模拟器 — MVP规格

## 它做什么（一句话）
让一个保定农户在 5 个回合里做取暖决策，并看到收入、取暖成本、合规与排放变化，最终得到成功转型或经济破产结局（回合2持续不整改可被处罚）。

## 给谁用（一个具体用户）
一个要做论文和课堂展示的本科生（你），用它向老师和同学解释“为什么清洁取暖在现实里会卡在可负担性”。

## 核心流程（用户走的3-5步）
1. 填写初始家庭信息（收入、盈余、面积、当前取暖方式、冬季取暖花费）。  
2. 回合1做开局选择：继续现状或准备转型。  
3. 回合2处理合规压力：可继续不整改（可能因连续违规被处罚）或申请整改。  
4. 回合3-4选择技术路线与增收/降耗策略，系统实时更新指标。  
5. 回合5固定触发“补贴减少”，系统输出终局（成功转型/经济破产）和政策诊断总结。

## 不在 MVP 里（「以后再说」清单）
- 多角色博弈（政府、燃气企业、村干部）。  
- 多地区对比与地图。  
- 真实账单/户级面板数据自动接入。  
- 长周期设备寿命与维修模型（如 8 年换炉）。  
- 多种随机冲击库（寒潮、限购、执法检查）作为正式模式。

## 怎么收集反馈
- 每次演示后记录 3 个问题：哪里不真实、哪里看不懂、哪个决策最难。  
- 让 3-5 位同学各跑一局，收集他们的最终结局与“为什么这么选”。  
- 每周迭代一次默认参数和事件文案，保留版本说明用于论文附录。
