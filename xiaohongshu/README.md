# 小红书宣传图

## 用法

### 方式一：直接发已导出的 PNG

`exports/` 目录下有 6 张 3:4 竖图，可直接上传小红书：

| 文件 | 内容 |
|------|------|
| `01-cover-qiuzhu.png` | 封面：求助 / 我做了一个网页 / 算算取暖账 |
| `02-stats.png` | 四格关键数据 |
| `03-how-to-play.png` | 怎么玩（三步流程 + 网址） |
| `04-identities.png` | 不同身份怎么参与 |
| `05-why-it-matters.png` | 为什么值得关注 + 报道引用 |
| `06-share-cta.png` | 拜托转发 + 征集经历 |

### 方式二：本地重新生成 / 改文案

1. 用浏览器打开根目录旁的 **`promo-images.html`**
2. 修改 HTML 里的文字或样式
3. 点击每张图下方的 **「下载 PNG」**（1080×1440）

> 下载依赖页面内嵌的 html2canvas CDN；若失败，可对预览区域截图。
